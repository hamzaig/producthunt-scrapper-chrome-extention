# 🚀 Quick Installation Guide

## Install in 3 Minutes!

### Step 1: Open Chrome Extensions
```
Type in address bar: chrome://extensions/
```
OR
```
Menu (⋮) → Extensions → Manage Extensions
```

### Step 2: Enable Developer Mode
Toggle the switch in **top right corner** ↗️

### Step 3: Load the Extension
1. Click **"Load unpacked"** button
2. Navigate to this folder:
   ```
   /Users/hamzakhalid/Professional/Helping Material/product-hunt-scrapper/producthunt-chrome-extension
   ```
3. Click **"Select"**

### Step 4: Pin to Toolbar (Optional but Recommended)
1. Click puzzle piece icon 🧩 in Chrome toolbar
2. Find "Product Hunt Scraper"
3. Click pin icon 📌

## ✅ Done!

You should now see the extension icon (🚀) in your toolbar.

## 🎯 First Use

1. **Go to** https://www.producthunt.com/
2. **Click** the extension icon (🚀)
3. **Click** "Scrape Products"
4. **Download** JSON when ready!

## 🐛 Issues?

### Extension not showing?
- Check if it's enabled in `chrome://extensions/`
- Look for errors (red text) on the extensions page
- Try reloading the extension (🔄 button)

### Can't find the folder?
Full path:
```
/Users/hamzakhalid/Professional/Helping Material/product-hunt-scrapper/producthunt-chrome-extension
```

### Still not working?
1. Close Chrome completely
2. Reopen Chrome
3. Go to `chrome://extensions/`
4. Check if extension is enabled
5. Try the extension again
